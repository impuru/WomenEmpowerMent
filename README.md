# Women Empowerment NGO Website

A React-based website for the NGO "Social Worker for Women Empowerment" founded by Babli Kumari from Muzaffarpur, Bihar.

## Features

- Responsive design using Material-UI (MUI)
- Navigation menu with Home, About, Services, Gallery, and Contact pages
- Banner section on the landing page
- Detailed information about Babli Kumari and the NGO
- Services offered by the organization
- Photo gallery with placeholder images
- Contact form and information
- Footer with email, address, and social media links

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository or download the source code.
2. Navigate to the project directory.
3. Install dependencies:

   ```
   npm install
   ```

### Running the Application

To start the development server:

```
npm start
```

The application will run on `http://localhost:3000`.

### Building for Production

To build the application for production:

```
npm run build
```
